# SMS Spam Detection – Machine Learning Pipeline

Repository contenente un notebook Google Colab dedicato alla rilevazione di SMS spam tramite tecniche di Machine Learning.

## Contenuto del progetto

- Caricamento e preprocessing del dataset (pulizia testo, normalizzazione, tokenizzazione)
- Feature extraction (Bag of Words / TF-IDF, Sentence-Trasformer)
- Suddivisione in training e test set
- Addestramento e confronto di diversi modelli di Machine Learning
- Valutazione tramite metriche standard
- Analisi comparativa delle performance dei modelli

## Obiettivo

Confrontare i principali algoritmi di classificazione testuale per identificare il modello più efficace nel distinguere SMS legittimi da messaggi spam.

Il notebook è strutturato in modo sequenziale e riproducibile, con commenti esplicativi per ogni fase della pipeline.
